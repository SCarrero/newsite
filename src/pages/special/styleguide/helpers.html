Array Helper Examples

Default

returns the value as an array (allowing for array functions like each, first, last, etc.)

{{#code 'html'}} \{{#each (array "" num) }}

\{{#if @first}}first item: \{{/if}} \{{.}} \{{#if @last}}- also last item\{{/if}}

\{{/each}} {{/code}} {{#each (array "" num) }}

{{#if @first}}first item: {{/if}} {{.}} {{#if @last}}- also last item{{/if}}

{{/each}}

concat

'concat' returns a new array containing items fromthe 2 arrays passed to it

{{#code 'html'}} \{{#each (array "concat" breakfast lunch)}} \{{this}}, \{{/each}}{{/code}}

{{#each (array "concat" breakfast lunch)}} {{this}}, {{/each}}

inArray

'inArray' returns true if array contains specified string

{{#code 'html'}} \{{#if (array "inArray" lunch "sandwich")}}

The lunch array contains "sandwich".

\{{else}}

The lunch array does not contain "sandwich".

\{{/if}}{{/code}} {{#if (array "inArray" lunch "sandwich")}}

The lunch array contains "sandwich".

{{else}}

The lunch array does not contain "sandwich".

{{/if}}

join

'join' joins elements of an array into a string with specified separators

{{#code 'html'}}

\{{array "join" breakfast " & "}}

{{/code}}

{{array "join" breakfast " & "}}

length

'length' returns the length of an array

{{#code 'html'}}

I have \{{array "length" breakfast}} items for breakfast

{{/code}}

I have {{array "length" breakfast}} items for breakfast

pop

'pop' removes the last item from the array and returns it (permanently modifies the array)

{{#code 'html'}}

Last item was \{{array "pop" lunch}}

The rest of lunch is: \{{#each lunch}} \{{.}} \{{/each}}

{{/code}}

Last item was {{array "pop" lunch}}

The rest of lunch is: {{#each lunch}} {{.}} {{/each}}

push

'push' returns the array with the new item added to the end (permanently modifies the array)

{{#code 'html'}}

The revised lunch is: \{{#each (array "push" lunch "muffin")}} \{{.}} \{{/each}}

{{/code}}

The revised lunch is: {{#each (array "push" lunch "muffin")}} {{.}} {{/each}}

randomize

'randomize' returns a copy of the array with the items in random order

{{#code 'html'}}

{{/code}}

The randomized lunch is: {{#each (array "randomize" lunch)}} {{.}} {{/each}}

reverse

'reverse' returns a copy of the array with the items in reversed order

{{#code 'html'}}

{{/code}}

The reversed breakfast is: {{#each (array "reverse" breakfast)}} {{.}} {{/each}}

shift

'shift' removes the first item from the array and returns it (permanently modifies the array)

{{#code 'html'}}

The rest of breakfast is: \{{#each breakfast}} \{{.}} \{{/each}}

{{/code}}

First item was {{array "shift" breakfast}}

The rest of breakfast is: {{#each breakfast}} {{.}} {{/each}}

slice

'slice' returns items from within the array, starting at nth item specified, up to an optional nth item

{{#code 'html'}}

{{/code}}

Items 2-3 of breakfast are: {{#each (array "slice" breakfast 1 3)}} {{.}} {{/each}}

sort

'sort' returns the array with the items in alphanumeric order (permanently modifies the array)

{{#code 'html'}}

{{/code}}
