Full-bleed bands

To create full bleed bands, you must define several nested <div> tags, starting from the outside and working inwards. Full bleed bands must go in the cross promo container in the TeamSite template.

  1. Background

    The outermost <div> is typically where you define the background for the band. Sometimes there are multiple bands that share a background.

  2. Landing Band

    Next, add a landing-band <div> to applies minimal left/right padding to prevent the 12-col grid content from touching the edge of the window at smaller screensizes. The 3 options vary depending upon how much padding is desired above/below the band contents.

    • Class .landing-band has the largest top/bottom padding
    • Class .landing-inner-band has a smaller amount of top/bottom padding
    • Class .landing-band-sides-only has no top/bottom padding.
    • Additional padding can always be added inside the band, using gutter classes or an empty callout.

    Note: If there is just one band within one background, you can apply the landing-band class to the outermost div with the background to avoid nesting.

  3. Row Container

    Next, add a <div> with a row-container based upon the maximum width that is desired for the content. Note: at screen widths below 1000px, all of these will look the same. The containers control how wide the 12-column grid can grow.

    • Class .row-container-narrow sets the max width of the contained rows to 1000px.
    • Class .row-container sets the max width of the contained rows to 1200px.
    • Class .row-container-wide sets the max width of the contained rows to 1300px.
    • Class .row-container-stretch sets the max width of the contained rows to 1500px.
  4. Rows

    Next, add the basic divs for the row(s). Do not apply left/right padding or margins to the rows.

  5. Columns

    Next, add the basic divs the column(s). Do not apply left/right padding or margins to the columns.

  6. Content

    Next, add in whatever content containers you need (cards, callouts, etc). Additional vertical or horizontal padding can be added as needed by using breakpoint specific gutter classes, small-to-default, small-to-large, empty callouts, etc.
